## 注意事项

- 首次建立索引需要一些时间，取决于文档数量和大小
- 索引文件会占用一定磁盘空间（通常是原文档的10-20%）
- 加密的PDF可能无法提取内容
- 图片中的文字无法识别（需要OCR功能）

## 故障排除

**问题：提示"索引不存在"**
- 解决：先运行 `python docfind.py index` 建立索引

**问题：某些PDF无法索引**
- 解决：可能是加密或扫描版PDF，尝试用其他工具转换

**问题：搜索结果不准确**
- 解决：使用更具体的关键词，或用引号搜索精确短语

## 性能优化建议

- 定期重建索引以保持最新
- 对于大量文档（>1000个），考虑分目录建立多个索引
- 搜索时使用具体关键词而非通用词汇
pip install -r requirements.txt
python docfind.py index ~/Documents  # 索引你的文档目录
python docfind.py search "关键词"     # 开始搜索
